## âš™ï¸ **Installation**
1. **Clone the repository:**  
   ```sh
   git clone https://github.com/your-username/EverDo.git
   cd EverDo
   ```
2. **Install dependencies:**  
   ```sh
   npm install
   ```
3. **Set up Firebase:**

   - Create your own `firebase.js` config file inside the `src/config/` directory.

   ```sh
   npx expo start
   ```
